Live Preview модулей (TopZurdo Menu)
====================================
Добавьте сюда PNG-скриншоты превью по ID модуля.
Имя файла: <moduleId>.png (например: fullbright.png, keystrokes.png).

Примеры:
- fullbright.png — до/после Fullbright
- keystrokes.png — как выглядит HUD клавиш
- coordinates.png — вид HUD координат

В будущем при наведении на карточку модуля будет показываться это превью.
